## Implementation

### New Scanner WAF Module (`internal/scanner/attacks/waf.go`)
- 97 HTTP attack requests across 5 categories:
  - Encoding bypasses (double URL, UTF-7, IBM037, fullwidth unicode, mixed case, comment injection, null byte)
  - HTTP request smuggling (CL.TE, TE.CL, TE.TE obfuscation, chunked fragmentation, double CL)
  - Parser confusion (Content-Type switching, multipart, parameter pollution, path confusion, method override)
  - CVE-specific (CVE-2024-1019, CVE-2025-29914, CVE-2026-21876, CVE-2022-48279, CVE-2023-24021)
  - Evasion techniques (SQL alternatives, XSS variants, path traversal variants)
- 9 raw TCP resource exhaustion attacks (JSON depth bomb, ReDoS, header flood, WebSocket flood)
- Implements both `AttackModule` and `RawTCPModule` interfaces

### New Encoding Functions (`internal/scanner/evasion/encoding.go`)
- `UTF7Encode()` — UTF-7 charset encoding
- `IBM037Encode()` — IBM EBCDIC encoding
- `HTMLEntityEncodeWithLeadingZeros()` — CVE-2025-27110 exploit
- `OverlongUTF8Encode()` — Overlong UTF-8 sequences
- `IISUnicodeEncode()` — IIS-style %uXXXX encoding
- All integrated into nightmare evasion mode

### New Scanner Profile (`waf-buster`)
- Concurrency: 30, Rate: 100 req/s, Timeout: 30s
- No crawling (go straight to attacks)
- Modules: waf, slowhttp, breakage, protocol, h3, owasp, injection
- Evasion: nightmare mode (all encodings)
