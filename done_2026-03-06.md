- **Go, Nginx, Flask showed zero errors** — goroutine/event-driven models handle load naturally
- **Current attacks are HTTP-level** — servers parse/reject them normally. True destruction needs raw TCP socket attacks
- **Glitch server survived self-scan** — adaptive blocking correctly identified and blocked the scanner
