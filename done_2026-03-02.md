## Task 3: Settings Persistence Tests (PR #23)

25 tests proving all server settings survive export/import cycles (simulating restarts).

### Bugs Found & Fixed
`ImportConfig()` only handled JSON-deserialized types, not native Go types from `ExportConfig()`:
- `int` values from `AdminConfig.Get()` silently dropped (all integer configs like max_labyrinth_depth, header_corrupt_level, etc.)
- `bool` values from `AdminConfig.Get()` silently dropped (protocol_glitch_enabled)
- `map[string]bool` from `VulnConfig.Snapshot()` silently dropped (all vuln group settings)

These bugs were masked in file-based persistence (JSON normalizes types) but broken for direct programmatic use.
